# Dead Dog Studios - One-Page Site

This is the source code for the Dead Dog Studios marketing website, hosted as a one-page application with a dedicated portfolio and use-cases page.

## Deployment
This repository is the canonical source of truth for the site. DigitalOcean App Platform is configured to deploy directly from this `master` branch.

## Navigation
- `index.html`: Main landing page with CTO services, AI capabilities, and case studies.
- `portfolio.html`: Detailed list of selected engagements and work.
- `use-cases.html`: Current small projects and active development items.
- `styles.css`: Global site styles.
- `script.js`: Site scripts.

## Syncing
Changes made to this repository are periodically synchronized with the `furry-road` master repository to ensure consistency across deployment environments.
